# swagger

Swagger

## Quick start

* create venv (python 3.8.5)
* Install requirements:
  ```
  cd swagger/
  pip install -r requirements.txt
  ```
* create superuser
  ```
  python manage.py createsuperuser
  ```
* Run
  ```
  python manage.py runserver
  ```
* Log in
* Swagger page available on your ```localhost``` ```/docs/``` page
